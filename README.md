# Web-Final-Project-Example-API
學校的網頁程式設計期末專案的範例資料 API

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
